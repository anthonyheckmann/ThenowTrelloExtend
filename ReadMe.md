# 说明
Thenow Trello Extend是一个Tampermonkey脚本，用于增加trello.com看板的功能

# 功能
1. 卡片图片快捷开关
2. 显示列表卡片数量
3. 在制品数量限制提示（在列表标题中增加“[在制品数]”）
4. 高亮显示所属人员（在卡片标题中增加“\`所属人员\`”）
5. 高亮显示分类（在卡片标题中增加“{分类名称}”）
6. 显示标签文字内容
7. 自定义看板背景图片
